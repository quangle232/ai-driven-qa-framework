Regression Execution Flow
End-to-end run on Jenkins  ·  trigger → setup → tests → report
Run Trigger
qa-agent script
Jenkins Pipeline
Scheduled / cron
1  ·  Checkout & Install
git checkout BRANCH
fetch latest code at the chosen branch
yarn install
install Node deps for this build
npx playwright install
ensure browsers are present
2  ·  Global Setup  ·  one-time auth
Sign in to the SUT
headed Chromium — handles 2FA / OTP if asked
Open Desk Sandbox
verify "Sandbox Mode" badge is visible
Warm CRM session
"Verify your identity" → password → Confirm
💾 Save storageState
.auth/storage-state.json (Desk + CRM)
3  ·  Test Run  ·  reuses storageState
playwright test --grep TAGS
each test starts already-logged-in (no per-test login)
Spec → POM → ActionKeyword
→ Playwright → Chromium → the SUT
test.step + expect.soft · retry × 3
on failure: friendly error + screenshot + trace + video
4 · Reporting & CI Feedback
Playwright Report
HTML & JSON — trace, screenshot & video retained on failure
Allure Report
Interactive report · single-file HTML · per-build link
Jenkins E-mail
Pass / fail summary & matrix · built by collect-playwright-stats.js
5 · Build Result
✅ SUCCESS
all selected tests passed
⚠️ UNSTABLE
only @bugs (known defects) failed
❌ FAILURE
real test fail OR global-setup error
